What are Interpreters?

Interpreters transform raw files into structured graph nodes. Each interpreter handles specific file types (CSV, JSON, Python, etc.), extracts metadata, and writes File nodes with properties and relationships to the graph. Interpreters are connection-agnostic — they work with files from local storage, rclone mounts, or any other source.
