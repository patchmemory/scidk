General-purpose Cypher queries and Python scripts for graph operations, data transformations, and automation.

What are Scripts?

Scripts are flexible execution modules that can perform any graph operation: data imports, schema migrations, batch updates, custom algorithms, or administrative tasks. Unlike analyses (which focus on read-only computations), scripts can modify the graph structure and are run on-demand rather than automatically.
