What are Scripts?

Scripts are flexible execution modules that perform graph operations, data transformations, analysis computations, file interpretation, or link creation. This registry includes all script types: analyses (read-only computations), interpreters (file parsing), links (relationship creation), and general-purpose automation. Scripts run on-demand and can modify the graph structure.
