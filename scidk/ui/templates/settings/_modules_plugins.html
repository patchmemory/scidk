What are Plugins?

Plugins extend SciDK's capabilities by registering custom API endpoints, defining new label types, and integrating with external services. Each plugin can register HTTP endpoints, publish custom labels to the graph, and execute arbitrary logic. Plugins provide a flexible extension mechanism without modifying core system code.

Plugin-Registered API Endpoints

Endpoints automatically registered by installed plugins. These cannot be edited manually.
