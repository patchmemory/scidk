What are Links?

Links create relationships between nodes using matching rules, fuzzy string similarity, API data, or imported tables. Each link defines a matching strategy (exact, Levenshtein, Jaro-Winkler, etc.), threshold parameters, and writes relationship edges to the graph with confidence scores and provenance metadata. Links enable cross-referencing between datasets and entity resolution across data sources.

Matching Approaches:

  • Local matching (rapidfuzz): Fast, in-memory string comparison using Python. Best for small-to-medium datasets where all comparisons fit in RAM.
  • Graph matching (Neo4j APOC): Runs similarity functions directly in Neo4j using APOC procedures. Best for large graphs where data is already stored in the database.
  • Hybrid: Combines both approaches — uses APOC to filter candidates in the graph, then ranks them locally with rapidfuzz. Best for large graphs with high selectivity requirements.
