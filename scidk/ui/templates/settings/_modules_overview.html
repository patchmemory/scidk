Modules Overview

Reference guide for SciDK's module types and their contracts.

SciDK uses a modular architecture where different types of components handle specific tasks. Each module type has a well-defined contract specifying inputs, outputs, and runtime requirements.

Module Types

Interpreters

Transform raw files into structured graph data. Each interpreter handles specific file types (CSV, JSON, Python, etc.) and extracts metadata.

Authoring: Scripts page | Settings: Modules > Interpreters

Plugins

Extend SciDK with custom endpoints, label types, and external integrations. Plugins can register API endpoints and publish custom labels.

Authoring: Plugins page | Settings: Modules > Plugins

Links

Create relationships between nodes using matching rules, API data, or imported tables. Links enable cross-referencing between datasets.

Authoring: Links page | Settings: Modules > Links

Labels

Define custom node types with properties, constraints, and relationships. Labels provide schema structure for your graph.

Authoring: Labels page | Settings: Modules > Labels

Analyses

Run analytical scripts over your graph to extract insights, generate reports, or perform calculations.

Authoring: Scripts page | Results: Results page | Settings: Modules > Analyses

Scripts

General-purpose Cypher queries and Python scripts for graph operations, data transformations, and automation.

Authoring: Scripts page | Settings: Modules > Scripts
